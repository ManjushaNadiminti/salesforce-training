# Day 2 - Salesforce Platform Basics

## What is Salesforce Platform?

Salesforce Platform is a cloud-based platform used to build and manage business applications.

---

# What is an App?

An App in Salesforce is a collection of objects, tabs, and tools used for a specific business purpose.

Example:
College Management App

---

# What is an Object?

An Object is a database table used to store information in Salesforce.

Examples:
- Student
- Faculty
- Account
- Contact

---

# What is a Tab?

A Tab is used to open and access objects and records in Salesforce.

---

# Difference Between App and Object

An App is a collection of tools and objects, while an Object stores data.

---

# What is Multi-Tenant Architecture?

Multi-tenant architecture means many companies use the same Salesforce platform while keeping their data secure and separate.

---

# Configuration vs Coding

## Configuration (No Code)

Used when tasks can be done using drag-and-drop tools.

Examples:
- Creating fields
- Reports
- Workflows

## Coding (Apex)

Used for advanced customization and complex business logic.

Examples:
- API integration
- Complex automation

---

# How Salesforce Developers Extend Functionality

Salesforce developers use Apex, APIs, Lightning Components, and automation tools to customize applications and add new features.

---
