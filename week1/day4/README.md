# Day 4 - Flow Builder and Automation

## What is Flow Builder?

Flow Builder is a no-code automation tool in Salesforce used to automate business processes and workflows.

---

# Why Automation Matters

Automation reduces manual work, saves time, improves accuracy, and increases productivity.

---

# Types of Flows

## Screen Flow

Screen Flow interacts with users using screens and forms.

Example:
Student registration form.

---

## Record-Triggered Flow

Record-Triggered Flow runs automatically when a record is created or updated.

Example:
Automatically send email after student registration.

---

# Automation Ideas for College Management System

## 1. Auto Email After Registration
Automatically send confirmation email to student.

## 2. Generate Student ID Automatically
System creates unique student ID.

## 3. Notify Faculty When Course is Full
Faculty receives automatic notification.

## 4. Send Fee Reminder
Students receive reminders before fee deadline.

## 5. Update Remaining Seats Automatically
System updates available course seats.

---

# Flow Design Example

## Process:
Auto email after student registration

### Trigger
New student record created

### Steps
- Check student email
- Generate confirmation message
- Send email automatically

### Final Action
Student receives confirmation email

---

# Manual vs Automated Process

## Manual Process
Staff manually sends emails and updates records.

## Problems
- Time consuming
- Human errors
- Delays

## Automated Process
Salesforce automatically performs tasks using flows.

## Benefits
- Faster process
- Better accuracy
- Saves time

---

# Reflection

Companies automate repetitive processes to improve productivity, consistency, speed, and accuracy while reducing manual effort and errors.
